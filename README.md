# FNP-Sales-Analsis
Interactive FNP Sales Analysis Dashboard built in Microsoft Excel to analyze revenue, customer behavior, product performance, and sales trends.
Objective

The goal of this project is to transform raw sales data into meaningful insights that support business decision-making and identify growth opportunities.

## Key Features
Revenue analysis across different occasions and months
City-wise order distribution and top-performing locations
Product-level performance and top revenue contributors
Category-wise revenue breakdown
Time-based analysis of order patterns
Key KPIs including total revenue, total orders, average delivery time, and average customer spend

## Key Insights
Sales are strongly influenced by seasonal and occasion-based demand
A few cities contribute significantly to overall order volume
Certain product categories dominate revenue generation
Sales patterns vary across months and peak during specific periods
Customer purchasing behavior follows distinct time-based trends

## Tools and Technologies
Power BI (Data Visualization and Dashboard Development)
Microsoft Excel / CSV (Data Source)
Data Cleaning and Transformation
Exploratory Data Analysis (EDA)

## Dataset
The dataset consists of transactional sales records including:
Order details (Order ID, Order Date, Delivery Date)
Customer information
Product details and categories
Revenue, cost, and profit metrics
Geographic data (City, State, Region)

## Conclusion
The dashboard provides a comprehensive overview of sales performance and highlights key areas for optimization. It enables stakeholders to make informed decisions by identifying trends, high-performing segments, and potential areas for improvement.
